# Dog Breed Viewer

A modern single-page React application that allows users to browse dog breeds and view random images for each breed. Built with React, TypeScript, and Vite, featuring a glassmorphism UI design with CloudSmiths branding.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/princessmistry3/breed-portal.git
cd breed-portal
```

2. Install dependencies:
```bash
npm install
```

## Running the Application

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173/`

## Running Tests

Execute the test suite:
```bash
npm test
```
## Usage

1. The application loads with a searchable dropdown of all dog breeds
2. Type in the search field to filter breeds in real-time
3. Click on a breed to view 3 random images
4. Select a different breed to see new images
